# Patient Management System

## Developed By
Madhu Sah

## Description
Patient Management System is a REST API application developed using Spring Boot and MongoDB. It allows users to perform CRUD (Create, Read, Update, Delete) operations on patient records.

## Features
- Add New Patient
- View All Patients
- View Patient by ID
- Update Patient Information
- Delete Patient
- MongoDB Database
- REST API using Spring Boot

## Technologies Used
- Java 17
- Spring Boot
- Spring Data MongoDB
- MongoDB
- Maven
- Postman
- Git & GitHub
